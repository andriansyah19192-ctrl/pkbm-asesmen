📖 Panduan Format File Import Siswa

Kolom yang Dikenali Sistem:

📌 nisn / NISN — Nomor Induk Siswa
📌 nama / Nama — Nama Lengkap *
🔑 tanggal_lahir / tgl_lahir — Tgl Lahir (jadi password)
📌 kelas / Kelas — Program/Kelas
📌 jenis_kelamin / jk — L/P
📌 hp / telp — Nomor HP/Ortu
📌 alamat — Alamat Lengkap
📌 tahun_masuk — Tahun Masuk
📌 status — Aktif/Lulus/dll

* Kolom wajib ada  |  🔑 Digunakan sebagai password login siswa

🔑 Format Tanggal Lahir yang Diterima:

✅ 2007-05-20  (YYYY-MM-DD)
✅ 20/05/2007  (DD/MM/YYYY)
✅ 20-05-2007  (DD-MM-YYYY)
✅ 20052007  (DDMMYYYY tanpa pemisah)
✅ 20 Mei 2007  (Teks Indonesia)
✅ Angka serial Excel (otomatis dikonversi)

Password login siswa = format DDMMYYYY, cth: 20052007

✅ Contoh Format CSV:

nisn,nama,tanggal_lahir,kelas,jenis_kelamin,hp,alamat
1234567890,Budi Santoso,2007-05-20,X IPS 1,Laki-laki,08123456,Jl. Merdeka No.1
0987654321,Siti Rahayu,2006-12-03,XI IPS 2,Perempuan,08198765,Jl. Pahlawan No.5

⚠️ Tips Penting:

  • Baris pertama file harus berisi nama kolom (header)
  • Kolom tanggal_lahir otomatis dijadikan password akun siswa
  • Format tanggal di Excel: pastikan kolom bertipe Text bukan Date untuk menghindari konversi otomatis, atau gunakan format YYYY-MM-DD
  • File Excel (.xlsx/.xls) bisa langsung diimport tanpa konversi
  • Download template untuk mendapatkan format yang sudah benar

📖 Panduan Format File Soal

📄 Format Word (.docx)

Tulis soal menggunakan format berikut (sistem mendeteksi otomatis):

1. Siapakah presiden pertama Republik Indonesia?
A. Soekarno
B. Soeharto
C. Habibie
D. Megawati
Jawaban: A

2. Pancasila disahkan pada tanggal...
A. 17 Agustus 1945
B. 18 Agustus 1945
C. 1 Juni 1945
D. 20 Mei 1945
Jawaban: C
Pembahasan: Pancasila disahkan PPKI pada 18 Agustus 1945

✅ Nomor soal: 1. atau 1)

✅ Pilihan: A. B. C. D. (bisa juga a. b. c. d.)

✅ Kunci: Jawaban: A atau Kunci: A

✅ Pembahasan: Pembahasan: ... (opsional)

📕 Format PDF

Sama dengan format Word. PDF yang diterima dengan baik:

✅ PDF teks (bukan scan/gambar)

✅ Soal pilihan ganda dengan format A/B/C/D

✅ Ada baris "Jawaban:" untuk kunci

⚠️ PDF scan/gambar: teks tidak bisa diekstrak otomatis

🔗 Google Form

Cara mendapatkan link yang tepat:

  1. Buka Google Form yang berisi soal
  2. Klik tombol "Kirim" → salin link form
  3. Atau salin URL dari address bar saat edit form
  4. Tempel di kolom URL di tab Google Form
Format URL yang valid:
https://docs.google.com/forms/d/e/1FAIpQL.../viewform
https://docs.google.com/forms/d/1FAIpQL.../edit
