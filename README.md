# 🎓 Sistem Asesmen PKBM Cendikia Cemerlang

Sistem manajemen asesmen berbasis web untuk PKBM Cendikia Cemerlang.

## 🌐 Akses Aplikasi

**[Klik di sini untuk membuka aplikasi →](https://USERNAME.github.io/pkbm-asesmen/)**

> Ganti `USERNAME` dengan username GitHub Anda

## ✨ Fitur

- 👨‍🏫 **Portal Admin** — Kelola siswa, guru, mata pelajaran
- 📝 **Manajemen Soal** — Buat & kelola bank soal
- 📊 **Asesmen & Penilaian** — Input dan rekap nilai siswa
- 📈 **Laporan & Raport** — Cetak laporan kemajuan belajar
- 🔑 **Multi-Role Login** — Admin, Guru, Siswa
- 🏫 **Naik Kelas** — Proses kenaikan kelas otomatis

## 🚀 Cara Deploy

1. Fork / clone repository ini
2. Aktifkan GitHub Pages di Settings → Pages → Branch: `main`
3. Akses di `https://USERNAME.github.io/pkbm-asesmen/`

## 🔐 Login Default

| Role | Username | Password |
|------|----------|----------|
| Admin | `admin` | `admin123` |
| Guru | (lihat data di aplikasi) | — |
| Siswa | (lihat data di aplikasi) | — |

## 📌 Catatan

- Semua data tersimpan di **localStorage** browser
- Tidak memerlukan server atau database
- Bisa digunakan offline setelah pertama kali dimuat
